# ContactHub

## Descripción

ContactHub es una aplicación web desarrollada con Python y Flask para la gestión de contactos. 
Permite registrar, buscar, editar y eliminar contactos almacenados en un archivo Excel mediante la librería OpenPyXL.

## Tecnologías utilizadas

* Python
* Flask
* HTML
* CSS
* JavaScript
* OpenPyXL
* Excel

## Funcionalidades

* Inicio de sesión
* Agregar contactos
* Buscar contactos
* Editar contactos
* Eliminar contactos
* Ordenar contactos de A a Z
* Generar reportes
* Validación de datos

## Instalación

1. Clonar el repositorio.
2. Instalar las dependencias:


pip install flask openpyxl


3. Ejecutar el proyecto:


python app.py


4. Abrir en el navegador:


http://127.0.0.1:5000
